# samkong
个人网站

这是一个由孔德晖（Sam Kong）开发的个人网站，主要用于展示其在教育科技领域的项目和成果。

## 关于作者

孔德晖是一位教育科技探索者，苹果杰出教育工作者（ADE 2023）及苹果专业培训讲师（APLS）。他专注于将人工智能和数字工具融入教育实践中，以提高学习效率。

## 网站结构

- 首页：介绍作者及其成就
- 作品集：展示各类教育项目
- 培训和分享：记录培训活动和经验分享
- 外出学习：学习访问和交流经历
- 资源：教育资源和材料
- 联系方式：联系方式和社交媒体链接

## 技术栈

- HTML5
- CSS3
- JavaScript
- Google Fonts
- 响应式设计
